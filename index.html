Supabase — Banco de dados em nuvem
Conecte ao Supabase para persistir todos os dados na nuvem
Credenciais do projeto
Encontre em: Dashboard → Settings → API → Project URL
Encontre em: Dashboard → Settings → API → anon public
Como configurar:
1. Acesse supabase.com e crie um projeto gratuito
2. No SQL Editor, execute o arquivo supabase_schema.sql para criar as tabelas
3. Cole a URL e a chave anon acima e clique em Testar e conectar
⚠ Coluna de fotos — SQL necessário:
Para salvar fotos de pacientes, profissionais e usuários, execute no SQL Editor do Supabase:
ALTER TABLE pacientes ADD COLUMN IF NOT EXISTS foto TEXT; ALTER TABLE profissionais ADD COLUMN IF NOT EXISTS foto TEXT;
